# What Did I Actually Agree To?

This project explores whether modern data permissions can be understood through interaction, onboarding, and progressive discovery — instead of long legal documents and one-click consent flows.

## 中文说明

本项目尝试探索：现代数据权限是否可以通过交互、引导与渐进式探索，变得比传统“长篇隐私政策 + 一键同意”更加可理解。

## Features

- Multi-screen permission world
- Route-based permission exploration
- Progressive disclosure on each permission page
- Calm motion and breathing room
- One language switch control
- Reflection page at the end of the journey

## Run locally

npm install
npm run dev
